Shout In The Dark is a whisper clone co-developed with Mike Semko. It's built entirely on the MEAN stack. We modularized the HTML with Angular v1.x, CSS with SCSS, and JS with Node. We ran Mongoose/Mongo with Node/Express on the backend. While we both worked on all facets of the project, I handled much of implementing the front-end Angular GUI and UX/UI. We used Gulp as our task runner, Git for version control, Github for issue tracking and management.

Reframe is a client who creates video and other media content for major brands. I worked with their designer to implement an elegant, and minimalist site that features nuanced UX with a not so nuanced JavaScript implementation. I used Gulp as my task runner and Git as my version control. I coded the HTML and SCSS from scratch and used jQuery for much of the JavaScript heavy lifting.

The Quote Today Is is a twitter account run by my Twitter bot. It posts random quotes several times a day implementing a quote database api. The bot recursively searches for a quote that fits within the character limits of twitters. This was built using JavaScript (Node) and deployed on Heroku.
